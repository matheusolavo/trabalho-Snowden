Reflexão sobre o filme Snowden

Depois de assistir ao filme Snowden, a gente entende o quanto os dados pessoais precisam ser tratados com cuidado. Qualquer descuido pode colocar a privacidade de milhares de pessoas em risco. Para quem trabalha como desenvolvedor front-end, lidar com dados sensíveis não é só uma questão de aparência ou funcionamento do site — é uma questão de responsabilidade. Informações como CPF, e-mail, endereço e senha podem facilmente vazar se não houver atenção. Por isso, é essencial usar HTTPS, proteger os dados com criptografia e sempre deixar claro para o usuário como essas informações serão usadas.

📝 Exemplo de Formulário

✅ Dicas Importantes para Cuidar dos Dados dos Usuários

  • 🔒 Use HTTPS (cadeado no site)
    Isso protege os dados que a pessoa envia no site, como nome e senha.
  • 📢 Informe o usuário sobre o uso dos dados
    Mostre para o usuário por que você está pedindo essas informações e como vai usá-las.
  • 🚫 Não guarde dados importantes no navegador
    É perigoso deixar coisas como CPF ou senha salvas ali, pois outra pessoa pode acessar.
  • 🔐 Proteja as senhas no sistema
    Mesmo que alguém invada, a senha estará embaralhada e não poderá ser usada.
  • 👁️ Não mostre dados pessoais na tela ou no link
    Evite mostrar coisas como o CPF ou e-mail na tela ou no endereço do site.
  • ✅ Verifique os dados antes de usar
    Isso evita que alguém envie códigos maliciosos que possam prejudicar o site.
  • 📄 Deixe a política de privacidade visível
    Assim o usuário pode ler como seus dados vão ser protegidos.
